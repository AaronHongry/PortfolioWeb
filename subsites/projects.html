Mocking Spongebob

October 1, 2018
Java

This is a very simple program created to take a set quantity of texts as an input and outputs it in alternating uppercase and lowercase. The program takes into account punctuation and will flip letters. This was originally created for Dmoj; an online platform with programming problems, however I kept this program because of the overwhelming use of the meme throughout late-2018 to early-2019.
